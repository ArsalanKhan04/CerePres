The Problem

Why EEGWriter?

EEGWriter is an AI tool for neurologists that analyzes EEG signals, highlights features and abnormalities, and generates structured reports.

  • Severe neurologist shortage globally
  • EEG review takes 20-40 minutes per study
  • Report writing is manual and time-consuming
  • Growing caseload needs automation

AI Models & Results

NeuroGate

Global Abnormality Classification
  • 90.70% accuracy (SOTA) on NMT
  • 90.20% on TUH Abnormal
  • Conv + Transformer architecture

NeuroTransformer

Fine-grained Event Detection
  • 83.17% on NMT-Events
  • Localized predictions per window
  • Identifies abnormality types

Application Demo

Technician Workflow

  1. Login with technician credentials
  2. Register new patient with demographics
  3. Upload EDF file for the patient
  4. System validates file format
  5. AI processing triggered automatically
  6. View processing status
Key: Glassmorphism UI, real-time status badges, toggle filters
Application Demo

Doctor Workflow

  1. View assigned patients and pending reviews
  2. Open patient case to see EEG files
  3. Launch interactive EEG visualizer
  4. Review AI-detected abnormalities
  5. Edit LLM-generated report draft
  6. Finalize and sign the report
Human-in-the-Loop: Doctor maintains full control over final diagnosis

Lessons Learned

Challenges & Solutions

Challenge
Long EEG files causing UI freezing during analysis
Solution
Async processing with Celery workers & Redis queue
Challenge
LLM generating inconsistent medical terminology
Solution
Custom prompt engineering with structured output format
Challenge
Real-time EEG visualization performance issues
Solution
Decimation, caching, and segment-based loading
Challenge
Different user workflows requiring distinct interfaces
Solution
Role-based routing with separate dashboard components
